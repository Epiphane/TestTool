I worked on the question table view of the question bank and the question response validation. The question table screen can be accessed by clicking the view questions button from the main toolbar. If you click the “New Question” button in the lower left hand corner, we currently hardcode in values for a new question and insert it only into the view; we have yet to insert the question into a database.
