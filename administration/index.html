Administration of the TestTool Project

The Test Tool allows a teacher to create and edit tests in a simple and structured way. It gives the test creator a large degree of freedom when creating tests-to create new questions, reuse old ones, and utilize many different question formats including multiple choice, short answer, and code input questions.

The primary intended user for the Test Tool is the Cal Poly Computer Science Department, so it is designed with the department's use in mind, including harder-to-find features like interactive test proctoring, semi-automated grading, and code testing.


This directory contains administrative files for the TestTool project.

  • Project group governance
  • Project work breakdown, for each team member
  • Inspection testing duty roster
  • Project meeting agendas
  • Project meeting minutes
  • Customer Questions
