TestTool Integration Test Plan


After the classes in each package are independently tested, perform integration testing in the following phases:

  1. Integrate the view package with the controller package and rerun the tests on the view package.
  2. Integrate the view package with the controller package and rerun the tests on the view and controller packages.
  3. Integrate the view, controller, and model package and rerun the tests on view, controller, and model packages.
  4. Add utils package to the integrated model, view, controller; run tests on the utils, model, view, controller packages.
  5. Run tests on the dao package.
  6. Add the dao package to the integrated model, view, controller; rerun tests on all packages.
