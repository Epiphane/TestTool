This design documentation describes the packages of the Test Tool software system. The system is comprised of three application programs:
  • the Test Tool program, for students to take tests and view scores
  • the Test Tool Instructor program, for instructors to create and edit tests. Instructors will also have access to student scores
  • the Test Tool database, which allows the above applications access to test, question, and user data
See the companion requirements specification document for complete details of the test tool functionality.
