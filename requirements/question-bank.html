2.5.1. List of Questions

When the instructor navigates to the question bank, he/she is presented with the screen in Figure 1.

Figure 1: Question Bank - Table.

The list of questions saved in the question bank is presented in a table view providing a brief overview of all the questions available. The instructor has the option of clicking a question to open it in the Question Editing screen, or selecting one or more questions and clicking the "Make Test" button to create a new test and open the New Test Options screen. If the raw list of questions is too overwhelming, the instructor can click the "Question Search" button in the lower left to open the screen shown in Figure 2.

Figure 2: Question Bank - Subjects.

Once in the search screen, the instructor can return to the table of questions by clicking the "Question Table" button in the lower left. The search question screen is organized into three tabs of increasing specificity. The screen is initially opened to the subjects tab, where the instructor can search or scroll for a subject. The screen also supports sorting the listed items by either item name, item creation date, or item modification date, as shown in Figure 3.

Figure 3: Question Bank - Sorted by Name.

Once the instructor clicks a subject, the test tab will be focused and tests belonging to the selected subject will be shown, and the subject name will be populated in the search bar. The instructor can also select the test tab manually, or clear out the search bar and click search to view all tests, as shown in Figure 4.

Figure 4: Question Bank - Tests.

Once again, the instructor can search or scroll for a test, and selecting a test will open the questions tab with the selected test name prefilled in the search bar, shown in Figure 5. Clearing the search bar and clicking search will remove the subject/test filter for the questions and will list all questions. The instructor once again can manually select the question tab to view all questions unfiltered.

Figure 5: Question Bank - Questions.

The instructor again has the option of selecting one or more checkboxes next to the questions to create a list of selected questions. Then, the instructor can click the "Make Test" button to create a new test using the selected questions. This button will take the instructor to the New Test Options screen. Clicking an individual question will open the Question Editing screen. Additionally, the instructor can use the question tab to search for all questions for a subject or a particular test using the search bar and appropriate keywords as shown in Figure 6.

Figure 6: Question Bank - Questions by Subject: "CPE 307".
