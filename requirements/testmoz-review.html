1.5.3. Testmoz

Testmoz is a tool that allows teacher to post a test to the web. It supports basic question types that testmoz automatically grades. When a teacher creates a test, testmoz requires a password for admin right to that test, and supplies you a url for the test. Anyone with access the the URL can take the test once it is publish. To take the test, navigate to the URL, provide your name, fill out test, and submit. The teacher will see the results for user each test taker’s provided name.

Good Features:

  • supports basic types of questions (true/false, multiple choice, multiple response fill in the blank)
  • supports automatic quiz grading
  • can export reports of student’s results to CSV
  • can randomize question order
  • can clone a test

Bad Features:

  • if the instructor changes the test while it is being administered, students must retake the entire test
  • student authentication is on the honor system tools
  • anyone can take the test (potential for spam)
  • no way for teachers to admin a test if they forget url or password

Missing Features:

  • no password recovery
  • no support for coding questions
  • no support for free response questions
  • no support for multiple test taking modes
