2.2.1 Creating a Test

This introductory scenario shows a user creating a simple test. Test creation is one of the most fundamental and commonly performed with the Test Tool. In this scenario the User is registered with the application and logged in to it. After the user selects the tests button on the landing page they are presented with the Test Overview Screen, seen in Figure 1.

Figure 1: The Test Overiew Page.

The Figure shows the highest level page involving test editing and creation. On the top menu bar the user is presented with a 'Create Test' button to create new tests, and a search bar to find already created tests. Here the screen is empty, because the user has yet to make anytests

To create a test the user selects the 'Create Test' button on the top menu bar or in the middle of the screen, which takes them to the test creation page shown in Figure 2.

Figure 2: New Tests Creation

This Figure has various data fields avaialable to the user. The first is a text field to name the test. The second and third will bring up a dialogue for the user to select a date that the test is intended to be taken on, seen in Figure 3.

Figure 3: Time and Date Picker.

This figure is simply a calendar which will auto-fill the data field it was brought up from when a time and date are selected

Next, selecting the "Assign To" field will produce a drop down menu that allows the instructor to pick which classes this test should be associated with. This menu is shown in Figure 4.

Figure 4: Assign To Menu

Finally, the user clicks the "next" button to go to the automatic creation page, seen in Figure 5.

Figure 5: Automated Test Generation

This screen provides the user with options on automated test generation. These options are the number of questions on the test, how many questions of each difficulty (easy, medium or hard), and the desired subjects. The screen with all finalized options is shown in Figure 6.

Figure 6: Finished Automated Test Generation

Finally the user clicks on the Generate Questions button to create the test.
