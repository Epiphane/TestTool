1.5.5. Schoolhouse Test

Schoolhouse Test
Schoolhouse Test is a product that provides a simple tabular interface to create and edit tests and test questions. It requires a $49 license to use the full features.
Good features:
  • Allows the teacher to edit test questions
  • Allows the teacher to edit placement of questions on a test
  • Allows the teacher to format the finished product
  • Provides a large variety of question formats
  • Provides a question bank to save and reuse test questions
Bad features:
  • Restricted to creating pencil-and-paper tests
Missing features:
  • No electronic test-taking
  • No automatic grading
  • No code input/validation
