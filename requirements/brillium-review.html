1.5.2. Brillium

Brillium is a web based test and survey creation tool. It provides a complete product for test creation, modifying, and analyzing. Because it is a web based platform, all tests and questions created are saved remotely, reducing the amount of local space needed as well as the chance of losing created materials. Brillium includes nearly all features outlined in this requirement, but noticeably does not include any sort of automated grading.

Overall Brillium is a good tool for creating tests and analyzing the results of them, but comes with a big price tag, with monthly pricing options starting at $49 a month for the personal option.

Good Features:

  • Includes all basic question types: Multiple Choice, True/False, Fill in, Essay.
  • Has built in question bank creation.
  • Generates “result sets” after tests are taking. Allows for ease of statistical analysis of individual questions (i.e. difficulty, times answered correctly, etc.)
  • Includes customer support and on-demand training videos for the software.
  • Relatively simple UI.
  • Keeps track of edits made to tests and questions (versioning).
  • Video Integration

Bad Features:

  • Tests are only available online for taking.
  • Half of the software is related to surveys, which may not apply as well in the academic test taking setting.
  • Limited number of accounts for professors and students unless enterprise version is purchased, which is expensive.
  • Entirely web based, must be online to work on tests.

Missing Features:

  • Automated grading
  • Does not include time limits on tests, only the time period that the test is available for taking. (Acceptable for proctoring mode, not acceptable for take home).
  • Question randomizing
  • Distinct operating modes
