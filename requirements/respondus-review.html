1.5.1. Microsoft Outlook

Respondus is a tool that allow users to create and manage exams. The tool is capable of creating exams that can be printed on paper or published to a supported learning management system. Some of the supported systems include: Blackboard, ANGEL, Brightspace by D2L, Pearson eCollege, and Moodle. Respondus can be used as a standalone application and the user can create offline exams. Although, it highly promotes its ability to integrate with other learning systems.
Respondus looks to be a good tool for users looking for learning systems integration. Although, the application only supports Windows operating systems.

Good Features:

  • integrates with popular learning management systems such as Blackboard and Moodle

  • changes to exams can be made offline

  • preview exams

  • add tables to a question

  • question type that require calculations

  • equation editor

Bad Features:

  • importing exams require specific formatting

  • when exporting an exam, the wizard shows server information. not useful for most users

  • embedded images are not displayed inline, only as text representation

Missing Features:

  • support for OSX and Linux operating systems

  • web application

  • add questions in bulk

  • option to add questions/answers that contain program code

  • editing exams do no sync across machines

  • does not auto generate exams

  • shared question bank
