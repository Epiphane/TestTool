Grade Individual Test

This scenario illustrates when an instructor grades a student's test. After a test has been created and taken by students, the instructor then grades the test. When an instructor selects grade from the Test Overview page, the instructor is shown the screen shown in Figure 1.

Figure 1: Teacher View of Grading a Test

  • Teachers are required to mark each question as correct or incorrect.
  • Correct/Incorrect are automatically populated with auto-graded responses. Teacher can change the grade for these questions by clicking radio buttons
  • Test Tool saves changes correct/incorrect instantly. Test Tool saves comments as the teacher types
  • Teacher can optionally add comments to student answers.
  • Teacher can navigate through questions by selecting question number, or by pressing the arrows next to the test numbers.
