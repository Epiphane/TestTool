2.3.2 View Test Questions

When a user clicks on an existing test from the "View Tests" page, a page with questions for that test are shown. This view is similar the student view of the test, the user can navigate through the pages and take the test.

The user can also click on the "Edit Test" button to make any changes. They will be taken taken to the "edit test" view.

Clicking on the numbers or the arrows at the bottom of the page will naviage the user through the question pages. The current view of the exam in Figure 1 is showing two questions per page.
