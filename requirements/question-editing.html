2.6.2. Editing Questions

When a question is selected for editing, the question is opened up in the screen shown in Figure 1.

Figure 1: Edit Question.

The instructor has the option to move the question to a different test using the subject and test drop down menus. The instructor also can change the type of question, as well the actual question text and the answer options and the correct answer. At the bottom of the page are three buttons that allow the instructor to cancel the edits, save the question as a new question and retain the original question, or save over the original question, which will prompt the user to confirm the overwrite.
