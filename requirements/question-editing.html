2.5.2. Editing Questions

The teacher teacher opens the question table for a test, selects a question, and clicks "Edit".

Figure 1: Edit Question.

The instructor has the option to move the question to a different test using the subject and test drop down menus as shown in Figures 2 and 3 respectively.

Figures 2 and 3: Edit Test and Subject.

Figure 4: Edit Question Type.

The instructor can use "Save as New Question" to leave the original question as is, then save this as a current question. The instructor can also click "Save" to overwrite the original question. Clicking "Cancel" will discard the changes and return the insturctor preview view, which is whichever view the instructor was in before the instructed started editing the question. If the instructor clicks "Save", TestTool with ask for a confirmation. This is because TestTool will prompt user for confirmation for actions that could cause data loss.
