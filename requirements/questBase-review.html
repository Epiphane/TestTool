1.5.4. questbase.com

Questbase is a web-based test-generation tool that supports many customizations and options. It is primarily intended for use by teachers for students, but also can create tests for public use. On of its nice features is that it has support for printing out the created test so it can be used in classroom setting. The biggest drawback of Questbase is that there is no way to create a test without creating an account first.

Good features:

  • Supports many different types of questions (Multiple Choice, True/False, Free Response etc.)
  • Supports importing questions from csv or excel files
  • Allows editing and reuse of old questions
  • Easily view test results
  • Ability to print test out and create physical copy
  • Ability for test creator to take test
  • Set password for test; donâ€™t allow random people to take test
  • Web-based

Bad features:

  • Need account to create tests
  • Can't easily create a test; too many options need to be chosen
  • UI is not very intuitive

Missing features:

  • Can't create proctored, monitored test
  • No easy test creation
