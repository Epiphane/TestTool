1.5. Related Systems

Widely used commercial systems that provide functionality comparable to the Test Tool includes:

  • Respondus

  • Brillium

  • TestMoz

  • QuestBase

  • SchoolHouseTech

  • EasyTestMaker

The goal of this test tool is to provide most of the core features found in the above tools. Although, this tool is not meant to be a consolidation of the above testing applications. Other noteworthy features that will be included in this test tool includes:

  • shared test bank

  • semi-automated test generation

  • semi-automated test grading

A detailed review of each related tool follows in these subsections:

  • Respondus Review

  • Brillium Review

  • TestMoz Review

  • QuestBase Review

  • SchoolHouseTech Review

  • EasyTestMaker Review


Following these reviews, a summary of the features is provided in a feature comparison matrix.
