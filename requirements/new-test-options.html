2.3.1. Creating New Tests



To create a new exam, the user can click on the 'Create New Test' button that is on the "View Tests" screen. Figure x is the screen that allows the user to choose initial options for the exam.

"Test Name" is the name of the exam that will be displayed to test takers.

Start and end time will make the test available at the specified times. The test creator has the option to disable the start and end time option. If disabled, the test will need to be manually enabled or disabled in the "View Tests" section.

The test can be assigned to all students taking a course, a specific section of a course, or a list of students.

The checkboxes in Figure x allows the test to be customized according to the description in the checkbox label.

To save all of the options for this test and move on to adding questions, the user can click on the "Create Test and Add Questions" button.
