1.5.1. EasyTestMaker

Easy Test Maker is a test-creation product that allows teachers to create tests with short answer, multiple choice, true/false, matching, and fill in the blank questions. The free version allows you to export your test to PDF and print, a $44.95/year license allows test questions and answers to be randomized, and a $74.95/year license also provides online test taking with automatic grading.

Overall, the product was not too difficult to use, but because I only had access to a free license I could not explore many of the features.

Good Features: Bad features: For free and pro licenses, tests are restricted to print-outs Missing features: No coding-related (scripting) question types No custom question types No time limit (on online tests) The only non-question format is Section Header (no video/paragraph options) No question bank Good Features:
  • Allows the teacher to add/edit test questions
  • Allows the teacher to reorganize the test questions and add headers
  • Provides a variety of question formats
  • Allows for copying questions from one test to another

Bad Features:

  • For free and pro licenses, tests are restricted to print-outs

Missing Features:

  • No coding-related (scripting) question types
  • No custom question types
  • No time limit (on online tests)
  • The only non-question format is Section Header (no video/paragraph options)
  • No question bank
